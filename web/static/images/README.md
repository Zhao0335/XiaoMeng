# 小萌头像 / 吉祥物图片

把你想用的 XiaoMeng 图片放到这个目录，文件名为 **`mascot.png`**（也可以是 `.jpg`、`.webp`、`.gif`），
它会出现在管理面板右下角浮动。

如果这里没有图片，面板会自动用一个内置的 SVG 卡通头像作为兜底，不影响功能。

## 推荐规格
- 正方形，最好 200×200 ~ 400×400 像素
- PNG 带透明背景效果最好
- 文件名严格为 `mascot.png` / `mascot.jpg` / `mascot.gif` / `mascot.webp`

> 如果想换其他位置/大小，改 `web/static/style.css` 里的 `.mascot` 规则即可。
