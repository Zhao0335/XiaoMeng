# 论文阅读技能

帮助用户高效阅读和分析学术论文，提取关键信息。

## 使用说明

当用户请求阅读论文时，按以下步骤操作：

1. **获取论文**：读取用户指定的论文文件
2. **提取结构**：识别标题、摘要、方法、结果、结论
3. **生成摘要**：用简洁的语言总结核心内容
4. **回答问题**：根据论文内容回答用户问题

## 核心规则

- 优先阅读摘要和结论，快速把握论文主旨
- 对于方法部分，提取关键创新点
- 对于实验部分，关注数据集和评估指标
- 用中文总结，保持专业但易懂

## 示例

用户请求: 帮我读一下这篇论文的主要贡献

```markdown
我来帮您分析这篇论文的主要贡献：

**论文标题**: [从论文提取]

**主要贡献**:
1. [贡献1]
2. [贡献2]
3. [贡献3]

**方法创新**: [简述方法创新点]

**实验结果**: [关键实验数据]
```

用户请求: 这篇论文用了什么数据集？

```markdown
根据论文内容，作者使用了以下数据集：

1. **[数据集名称]**: [规模和用途]
2. **[数据集名称]**: [规模和用途]

评估指标包括: [列出指标]
```
