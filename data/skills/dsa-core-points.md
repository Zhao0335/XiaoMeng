# 数据结构与算法核心考点整理

> 整理时间：2026-05-23 | 来源：网络资料汇总 + 小萌整理

## 一、线性表

### 顺序表
- 随机访问 O(1)，插入/删除 O(n)
- 动态扩容策略（翻倍扩容，均摊 O(1)）
- **考点**：插入/删除时元素移动次数计算

### 链表
| 类型 | 特点 |
|------|------|
| 单向链表 | 每个节点存 data + next |
| 双向链表 | 存 prev + data + next |
| 循环链表 | 尾节点指向头节点 |
| 静态链表 | 用数组模拟链表（游标实现） |

- **考点**：链表反转、合并有序链表、快慢指针找环/中点、删除倒数第K个节点
- 链表 vs 顺序表：插入/删除 O(1) vs O(n)，查找 O(n) vs O(1)

## 二、栈与队列

### 栈（FILO）
- 括号匹配、表达式求值（中缀转后缀）、函数调用栈
- **单调栈**：找下一个更大/更小元素，O(n)

### 队列（FIFO）
- **循环队列**：head/tail 指针，判空判满条件
- **双端队列（Deque）**：两端均可插入删除
- **单调队列**：滑动窗口最大值（O(n)）
- **优先队列**：基于堆实现

### 考点
- 用两个栈实现队列 / 用两个队列实现栈
- 单调栈解决接雨水、柱状图最大矩形

---

## 三、树与二叉树

### 二叉树遍历
| 遍历方式 | 顺序 | 实现 |
|----------|------|------|
| 前序 | 根→左→右 | 递归/栈迭代 |
| 中序 | 左→根→右 | 递归/栈迭代 |
| 后序 | 左→右→根 | 递归/栈迭代/双栈 |
| 层序 | 逐层从左到右 | BFS 队列 |

### 常见题型
- 根据前序+中序（或后序+中序）重建二叉树
- 二叉树的最大/最小深度
- 二叉搜索树（BST）：中序有序，插入/查找 O(log n)
- 平衡二叉树（AVL）：左右子树高度差 ≤ 1
- 最近公共祖先（LCA）
- **二叉堆**：完全二叉树，最大堆/最小堆，堆排序

## 四、图

### 存储方式
- 邻接矩阵：O(V²) 空间，适合稠密图
- 邻接表：O(V+E) 空间，适合稀疏图

### 遍历
- DFS（递归/栈，visited 标记）
- BFS（队列，求最短路径）

### 最短路径
| 算法 | 适用 | 复杂度 |
|------|------|--------|
| Dijkstra | 单源，非负权 | O(V²) 或 O(E log V) 堆优化 |
| Bellman-Ford | 单源，可负权 | O(VE) |
| Floyd-Warshall | 全源 | O(V³) |
| SPFA | 单源，负权优化 | 平均 O(E)，最坏 O(VE) |

### 最小生成树
- Kruskal：按边权排序 + 并查集，O(E log E)
- Prim：从点出发，贪心扩展，O(V²) 或 O(E log V)

### 其他
- **拓扑排序**：仅 DAG，BFS（入度为0）或 DFS
- **并查集**：路径压缩 + 按秩合并，近乎 O(1)
- 强连通分量（Tarjan/Kosaraju）
- 二分图判定（染色法）

## 五、查找

### 二分查找
- 有序数组，O(log n)
- 变体：找第一个/最后一个等于目标、找旋转数组最小值

### 哈希表
- 冲突解决：链地址法、开放地址法（线性探测/二次探测/双重哈希）
- 负载因子与扩容
- **考点**：两数之和、LRU Cache（哈希+双向链表）

### 平衡树
- AVL、红黑树（map/set 底层）、B树/B+树（数据库索引）
- 红黑树性质：根黑、叶黑、红子黑、黑高相同

## 六、排序

### 常见排序对比

| 算法 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|--------|
| 冒泡 | O(n²) | O(n²) | O(1) | ✅ |
| 选择 | O(n²) | O(n²) | O(1) | ❌ |
| 插入 | O(n²) | O(n²) | O(1) | ✅ |
| 希尔 | O(n^1.3~1.5) | O(n²) | O(1) | ❌ |
| 快排 | O(n log n) | O(n²) | O(log n) | ❌ |
| 归并 | O(n log n) | O(n log n) | O(n) | ✅ |
| 堆排 | O(n log n) | O(n log n) | O(1) | ❌ |
| 计数 | O(n+k) | O(n+k) | O(k) | ✅ |
| 桶排 | O(n+k) | O(n²) | O(n) | ✅ |
| 基数 | O(d(n+k)) | O(d(n+k)) | O(n+k) | ✅ |

### 考点
- 快排 partition 过程（单路/双路/三路）
- 第 K 大/小 → 快排 partition O(n) / 堆 O(n log k)
- 外部排序（归并思想，多路归并）

## 七、字符串

### KMP
- next 数组构建，匹配 O(n+m)
- 核心思想：利用已匹配信息避免回溯

### Trie（前缀树）
- 字符串检索、前缀匹配、词频统计
- 节点：child[26] + isEnd

### 其他
- 回文串（中心扩展 / Manacher 算法 O(n)）
- 字符串哈希（滚动哈希，Rabin-Karp）
- 最长公共子序列（LCS）、最长公共子串（DP）

## 八、动态规划

### 经典题型

| 题目 | 状态定义 | 转移方程要点 |
|------|----------|-------------|
| 斐波那契 | dp[i] = 第i项 | dp[i] = dp[i-1] + dp[i-2] |
| 爬楼梯 | dp[i] = 到第i阶的方法数 | dp[i] = dp[i-1] + dp[i-2] |
| 背包01 | dp[i][w] = 前i个物品容量w最大价值 | 选/不选 |
| 完全背包 | dp[w] = 容量w的最大价值 | 同物品可重复选 |
| LIS | dp[i] = 以i结尾的最长上升子序列长度 | dp[i] = max(dp[j]+1) |
| LCS | dp[i][j] = 前i前j的最长公共子序列 | s1[i]=s2[j]时+1，否则取max |
| 编辑距离 | dp[i][j] = 最小编辑次数 | 增/删/改 |
| 区间DP | dp[i][j] = 区间[i,j]最优值 | 枚举分割点 |
| 数位DP | dp[pos][state] | 按位枚举，记忆化搜索 |

### 优化技巧
- 滚动数组（空间降维）
- 四边形不等式（区间DP加速）
- 斜率优化/单调队列优化

## 九、经典算法思想

- **分治**：归并排序、快速排序、最大子数组和
- **贪心**：区间调度、哈夫曼编码、活动选择
- **回溯**：全排列、子集、N皇后、数独
- **双指针**：两数之和、盛最多水、三数之和
- **滑动窗口**：无重复最长子串、最小覆盖子串
- **位运算**：异或（找唯一出现奇数次的数）、lowbit、状态压缩
