# 日历管理插件

## 功能概述

本插件为 XiaoMeng 提供日历管理功能，可以：

- 向 calendar_backend 系统添加日程安排
- 管理待办事项
- 通过 QQ 渠道验证账号信息
- 支持多账号切换（每次切换需密码验证）

## 使用方法

### 命令方式

```
/日历 登录 {账号名} {密码}
/日历 切换 {账号名} {密码}
/日历 登出
/日历 账号列表
/日历 注册 {别名} {用户名} {密码}
/日历 添加日程 {标题} [日期] [时间] [地点]
/日历 添加待办 {标题} [截止日期] [优先级]
/日历 查看日程
/日历 查看待办
/日历 帮助
```

### 自然语言方式

你可以直接告诉小萌你的安排，她会自动识别并添加：

- "明天下午3点有个会议"
- "周五前要交报告"
- "下周一看牙医，在市中心医院"
- "后天晚上7点和朋友吃饭"

## 配置说明

编辑 `config.json` 文件：

```json
{
  "enabled": true,
  "backend": {
    "url": "http://127.0.0.1:5522"
  }
}
```

## 安全说明

- 密码使用 bcrypt 安全存储
- 支持登录尝试限制，防止暴力破解
- 账号切换必须验证密码
- 会话有有效期限制

**用户**: /日历 注册 我的账号 myuser mypassword123
