# Changelog 2026-05-23

## gateway.py 重构拆分

将 `core/qq/gateway.py`（原 2395 行）拆分为四个 Mixin 文件，提高可维护性。

### 新增文件

| 文件 | Mixin 类 | 内容 |
|------|----------|------|
| `core/qq/memory.py` | `MemoryMixin` | DB 初始化、消息存取、记忆压缩（`_init_db`, `_save_message`, `_compress_memory` 等 11 个方法）|
| `core/qq/identity.py` | `IdentityMixin` | 身份解析、权限判断（`_resolve_identity`, `_get_effective_level` 等 5 个方法）|
| `core/qq/prompt_builder.py` | `PromptBuilderMixin` | Prompt 构建（`_read_soul`, `_build_system_prompt`, `_build_messages`, `_inject_history_images`, `_build_router_prompt`）|
| `core/qq/task_runner.py` | `TaskRunnerMixin` | LLM 推理主循环（`_make_task_coro_factory` 整体，约 500 行）|

### 修改文件

- `core/qq/gateway.py`：~1068 行（原 2395 行，减少 55%）
  - 改为继承 `QQGateway(MemoryMixin, IdentityMixin, PromptBuilderMixin, TaskRunnerMixin)`
  - 保留：`__init__`、事件处理、图片处理、主动发言、内心世界等核心逻辑
  - 删除：本地 HTTP 中继服务器（`_start_relay_server`，原 §三 功能，已废弃）

### 不影响功能

拆分使用 Python Mixin 模式，运行时 `self` 仍是 `QQGateway`，所有方法调用路径不变。

## 内心世界（Inner World）

新增 `core/inner_world/` 模块（平台无关）：
- `InnerWorldAgent` — 空闲时的自主活动代理
- `InnerWorldEventLogger` — 跨群事件流（events.jsonl）

## 记忆路由修复 + 主动记忆注入

### 记忆路由

- `core/qq/memory.py`（`_compress_memory`）：
  - **群聊**压缩摘要写 `data/memory/group_{group_id}.md`，不再写 MEMORY.md
  - **私聊**压缩摘要（含 person_facts）统一写 `data/memory/{identity}.md`，不再写 MEMORY.md
  - MEMORY.md 今后只接收 `add_memory(scope="global")` 的显式条目，增长大幅降低

- `core/qq/prompt_builder.py`（`_build_system_prompt`）：
  - 群聊场景自动加载 `data/memory/group_{group_id}.md`，注入"关于这个群的历史记忆"段落（最近 2000 字）

### 主动记忆注入

- `core/qq/task_runner.py`：每条消息路由后自动对用户消息做预搜索（≥8字），命中时追加"自动检索到的相关记忆"到 system prompt，减少记忆遗漏

- `core/qq/tools.py`（`_search_memory`）：SQLite 查询 keys 加入 `"global"` 作用域，`add_memory(scope="global")` 的条目现在可被 search_memory 命中

### MEMORY.md 后台压缩

- `core/qq/gateway.py`：新增 `_memory_md_compressor_loop` 后台任务（每 30 分钟检查），MEMORY.md 超过 8000 字时用 CLOUD 模型二次压缩，保留规则/设定，提炼/删除冗余内容，维持文件 ≤ ~6000 字

## Local 模型提示词优化

- `core/qq/prompt_builder.py`：LOCAL 模型 system prompt 改用更直接的称谓指引（"你正在和主人哥哥直接对话，直接对他说话"），避免小萌将对话对象当成第三人称引用
- `core/inner_world/events.py`：内心世界事件压缩提示词改用 system_prompt + 简短 user message 格式，并将 SOUL.md 前段内容注入为人设上下文，max_tokens 降至 30，输出过长则丢弃
